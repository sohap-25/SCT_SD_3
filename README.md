# SCT_SD_3

## Sudoku Solver

A Java-based Sudoku Solver application that automatically solves Sudoku puzzles using the Backtracking algorithm.

## Technologies Used
- Java
- VS Code

## Features
- Solves Sudoku puzzles automatically
- Uses Backtracking algorithm
- Accepts user input for unsolved Sudoku grid
- Displays solved Sudoku puzzle

## How to Run

1. Open the project in VS Code

2. Compile the program:

```bash
javac SudokuSolver.java
```

3. Run the program:

```bash
java SudokuSolver
```

4. Enter the Sudoku puzzle row by row using:
- Numbers 1 to 9 for filled cells
- 0 for empty cells

## Example Input

```text
3 0 6 5 0 8 4 0 0
5 2 0 0 0 0 0 0 0
0 8 7 0 0 0 0 3 1
0 0 3 0 1 0 0 8 0
9 0 0 8 6 3 0 0 5
0 5 0 0 9 0 6 0 0
1 3 0 0 0 0 2 5 0
0 0 0 0 0 0 0 7 4
0 0 5 2 0 6 3 0 0
```

## Output
The program displays the solved Sudoku puzzle automatically.
